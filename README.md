# Pixel Plane Shooter

A fast, arcade-style pixel airplane shooter you can play in the browser.

Fly, dodge, and fire through waves of incoming enemies.  
The controls are simple, the pace keeps rising, and every run pushes you to beat your best score.

## Why try it

- Instant play, no install needed
- Pixel-art style with lightweight effects
- Easy controls, quick restarts, addictive score chase

## Controls

- `Arrow Keys` / `WASD`: Move
- `Space`: Shoot
- `R`: Restart after game over

## Built with

- `HTML`
- `CSS`
- `JavaScript` (Canvas API)

## Run locally

Open `index.html` directly in your browser.

## About this project

This is a personal practice project I built to improve my frontend/game-dev fundamentals, including:

- Real-time game loop
- Collision detection
- Input handling
- Difficulty scaling
- UI/HUD updates

If you have ideas for upgrades (boss fights, power-ups, sound effects, online leaderboard), feel free to open an issue or PR.
